# ChessPulse

ChessPulse is a product-style chess web app prototype for the nFactorial chess assignment.

It is designed to feel like a real consumer product rather than a plain chessboard demo: instant play, AI opponent, AI Coach feedback, local progress, city leaderboard, and a clear premium layer.

## Features

- Legal chess moves powered by `chess.js`
- AI opponent with simple tactical evaluation
- AI Coach feedback after player moves
- Player profile and city ladder concept
- Match stats saved locally in the browser
- Responsive one-file static app
- Ready for GitHub Pages, Vercel, Netlify, or any static hosting

## Run

Open `index.html` directly or serve the folder with any static server.

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Product positioning

ChessPulse is for casual chess players who want quick games and immediate improvement feedback. The product loop is simple: play a game, get coached, return to improve your rating and city rank.
